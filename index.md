### Download and Install latest release (v1.5.0)

```console
$ wget https://github.com/openucx/ucx/releases/download/v1.5.0/ucx-1.5.0.tar.gz
$ tar xzf ucx-1.5.0.tar.gz
$ cd ucx-1.5.0
$ ./contrib/configure-release --prefix=$PWD/install
$ make -j8 install
```
