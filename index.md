### Download and Install latest release (v1.6.1)

```console
$ wget https://github.com/openucx/ucx/releases/download/v1.6.1/ucx-1.6.1.tar.gz
$ tar xzf ucx-1.6.0.tar.gz
$ cd ucx-1.6.1
$ ./contrib/configure-release --prefix=$PWD/install
$ make -j8 install
```
